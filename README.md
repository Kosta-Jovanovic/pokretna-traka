# pokretna-traka

Pokretna traka je projekat u kom se kutije razvrstavaju po boji i materijalu. Kada se projekat pokrene korisnik ce imati izbor da li ce da razvrstava kutije po boji ili po materijalu, izbor ce se korisniku dati klikom na dugme. Traka ce raditi tako sto ce odgurati kutiju do kraja trake gde ce se nasloniti na "Soil Moisture Sensor" koji ce izmeriti provodljivost materijala i po tome ce odrediti od kog matrijala je napravljena kutija a iznad trenutnog polozaja kutije stoji "TCS3200 Color Sensor" koji ce odrediti da li je kutija obojena u crnu ili belu.
